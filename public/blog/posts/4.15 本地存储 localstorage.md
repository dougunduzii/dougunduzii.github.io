# 本地存储 localStorage

localStorage 是 HTML5 提供的一种在浏览器中存储数据的机制，数据以键值对的形式存储，没有过期时间。

## 基本语法

```javascript
// 存储数据
localStorage.setItem('key', 'value')

// 读取数据
const value = localStorage.getItem('key')

// 删除数据
localStorage.removeItem('key')

// 清空所有数据
localStorage.clear()
```

## 代码示例

```javascript
// 存储用户信息
localStorage.setItem('username', '张三')
localStorage.setItem('age', '25')

// 读取用户信息
const username = localStorage.getItem('username')
const age = localStorage.getItem('age')
console.log(username, age)  // 张三 25

// 存储对象（需要转换为JSON字符串）
const user = {
  name: '李四',
  age: 30,
  email: 'lisi@example.com'
}
localStorage.setItem('user', JSON.stringify(user))

// 读取对象
const storedUser = JSON.parse(localStorage.getItem('user'))
console.log(storedUser.name)  // 李四

// 删除单个数据
localStorage.removeItem('age')

// 清空所有数据
localStorage.clear()
```

## 特点

1. **持久性**：数据永久存储，除非手动删除或清除浏览器缓存
2. **容量限制**：通常为 5MB 左右
3. **同源策略**：只能访问同域名下的 localStorage
4. **字符串存储**：只能存储字符串类型，其他类型需要转换

## 注意事项

- localStorage 只能存储字符串，存储对象或数组时需要使用 `JSON.stringify()` 和 `JSON.parse()`
- 不要在 localStorage 中存储敏感信息
- 在隐私模式下，localStorage 可能会被禁用
